Project Overview


Redesigned application will help to solve the issues that Perk Payroll company has been facing with its low customer retention rate with clean layout




Problem Context


This company has been facing poor User Experience in the application.


The issue was with the categorisation of the administration of the payroll application.

  • The definitive roles like Employee, Manager and Administrator roles were poorly defined in the application.

  • This was due to the poorly structured Information Architecture which affected the mobility in the application while moving from one role (Employee) to another (Manager).

  • There was no difference in the dashboards of the application when you are the Manager or Employee of the company.



Solution


The solution is to redesign the application by creating separate profiles for Employee, Manager, Administrator And defining their roles and limitations.


The task was to structure the Information Architecture and clearly layout the difference in being a manager, employee And admin to the company.



Ideation


By considering problems once again and with a critical analysis the major problems was with the dashboards and mis-structured architecture to each of the profiles




Information architecture



To differentiate their roles in the new architecture the admin roles and duties have been segregated for Administrator, Manager


The goal here was to create a dashboard with intuitive features for a quick look up for the tasks that has the basic features.


  • Leave dashboard, Attendance overview
  • Log in/ log out options
  • Birthdays, events
  • Taxes, payroll, news and updates
  • Individual profiles

Design


To differentiate in dashboards of Employee and Manager, he/she gets two additional tabs Me, My team


  • Me option displays the Manager as employee profile.
  • My team displays the list of employees under manager and manager actions towards employees

To differentiate in dashboard of Administrator, he/she gets two additional tabs Me, Admin


  • Me option displays the Administrator as employee profile.
  • Admin displays the list of managers, employees under administrator and administrator actions towards them

In order to redesign the application it is divided into 3 modules, Employee, Manager and Admin with their actions performed.



Employee module


In the module, Employee has certain functions performed and necessary services in dashboard


Employee functions


  • Check attendance log in, log out options
  • View leave dashboard
  • View taxes
  • View news and updates

Employee services
